# Image Search Engine

A simple web-based image search engine that allows users to search for images using a search query. The application fetches images from an external API and displays them in a grid layout with options to view more results.

## Features
- Search for images by keyword
- Display search results in a responsive grid
- Show more button to load additional images
- Clean and modern UI

## Live Demo
Check out the live demo: [https://itsiamdev.github.io/Image-Search-Engine/](https://itsiamdev.github.io/Image-Search-Engine/)

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter a search term in the input box
2. Click the Search button
3. Browse the displayed images
4. Click "Show more" to load additional results
